# noname

## **Api scripts**

## Installation

```bash
$ yarn setup
```

### Running the app

```bash
$ yarn dev:api

# or

# go to api directory
$ cd api

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# go to api directory
$ cd api

# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## **web site scripts**

## Running the app

```bash
$ yarn dev:web

# or

# go to web directory
$ cd web

# dev
$ yarn dev
```
